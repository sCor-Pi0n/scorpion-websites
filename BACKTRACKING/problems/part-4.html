permutari_pfp

Cerință

Se citește un număr natural n (n<16). Afișați în ordine lexicografică toate permutările mulțimii {1,2,…,n} în care elementele pare sunt puncte fixe (nu își schimbă poziția).

Programul citește de la tastatură numărul n.

Programul va afișa pe ecran permutările cerute, câte una pe fiecare rând și având elementele separate prin câte un spaţiu.

Exemplu

Input

5

Output

1 2 3 4 5 ​
1 2 5 4 3 ​
3 2 1 4 5 ​
3 2 5 4 1 ​
5 2 1 4 3 ​
5 2 3 4 1​
​
Explicația exemplului

* 1 < n < 16

Explicația exemplului

Se vor genera permutările în care 2 și 4 nu își modifică poziția.

Cod (soluție)
solutie.cpp
#include <iostream>
using namespace std;
int n;
int a[20];
int v[20];
void afisare()
{ 
    for(int i=1;i<=n;i++)
        cout<<a[i]<<" ";
    cout<<"\n";
}
void back(int k)
{
    for(int i=1;i<=n;i++)
        if(!v[i])
    { 
        a[k]=i;
        v[i]=1;
        if(a[k]%2==1 || a[k]==k) 
        {
            if(k==n) 
                afisare(); 
            else back(k+1);
        }
        v[i]=0;
    }
}
int main()
{
    cin>>n;
    back(1);
    return 0;
}

Cerință

Programul citește de la tastatură cuvântul s.

Programul va afișa pe ecran pe rânduri separate anagramele cuvântului citit în care vocalele rămân pe pozițiile inițiale.

Exemplu

Input

 dorel

Output

doler
dorel
loder
lored
rodel
roled
Restricții și precizări

* cuvântul s are cel mult 11 caractere și este format din litere mici distincte

Explicația exemplului

Anagramele sunt afișate în ordine lexicografică, iar literele o și e rămân pe pozițiile inițiale.

        return 0;
    return 1;}
    void back(int k)

        if(!x[i])

solutie.cpp
#include <iostream>
using namespace std;
int n,a[11],v[10];
int ok(int k)
{
    if(a[1]==0) return 0;
    int cp=0;
    for(int i=1;i<=k;i++)
        if(a[i]%2==0) cp++;
    if(k==n && cp<=k-cp) return 0;
    return 1;
}
void afisare()
{
    for(int i=1;i<=n;i++)
        cout<<a[i];
    cout<<"\n";
}
void back(int k)
{
    for(int i=0;i<=9;i++)
        if(!v[i])
        {
            a[k]=i;
            v[i]=1;
            if(ok(k))
                if(k==n) afisare();
                else back(k+1);
            v[i]=0;
        }
}
int main()
{
    cin>>n;
    back(1);
    return 0;
}

Cerință

Se citesc două numere naturale n și m. Afișați în ordine lexicografică toate submulțimile cu m elemente ale mulțimii {1, 2, ..., n} în care nu există două elemente pare alăturate.

Programul citește de la tastatură numerele n și m.

Programul va afișa pe ecran submulțimile cerute, câte una pe fiecare rand și având elementele separate prin câte un spaţiu.

Exemplu

Input

5 3

Output

1 2 3 
1 2 5 
1 3 4 
1 3 5 
1 4 5 
2 3 4 
2 3 5 
3 4 5
Restricții și precizări

* 1 ≤ m < n ≤ 20

Explicația exemplului

Numrele 2 și 4 nu apar pe poziții alăturate.

Cod (soluție)
solutie.cpp
#include <iostream>
using namespace std;
int n;
int m;
int X[21];
void afisare()
{
    for(int i=1;i<=m;i++)
        cout<<X[i]<<" ";
    cout<<"\n";
}
void back(int k)
{
    for(int i=X[k-1]+1;i<=n;i++)
    { 
        X[k]=i;
        if(k==1 || X[k]%2+X[k-1]%2!=0)
        {
            if(k==m)
                afisare();
            else back(k+1);
        }
    }
}
int main()
{
    cin>>n>>m;
    back(1);
    return 0;
}
