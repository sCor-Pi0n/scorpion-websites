# Backtracking — Presentation Site (Multi-page)

## Conținut
- Home: `index.html`
- Problems: `problems.html`
  - `problems/part-1.html` ... `problems/part-6.html` (autori)
  - fiecare parte are 5 șabloane de probleme
- Theory: `theory.html` (include imaginile din `assets/exemplu-*.png`)
- Bibliography: `bibliography.html`

## Rulare
- Simplu: deschide `index.html`
- Recomandat: rulează un server local:
  - Windows: `py -m http.server 8000`
  - macOS/Linux: `python3 -m http.server 8000`
  - apoi: `http://localhost:8000`

## Editare
- Completează textele direct în fișierele `.html`.
- Pentru cod, editează blocurile din paginile de probleme (secțiunea „Cod (soluție)”).
